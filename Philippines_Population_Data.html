Note: After having an overview of the data, we have seen that the ‘Year’, ‘Growth Rate’, and ‘Population’ column is in another data type that might cause an error later on.

Convert ‘Year’ from Double to Date

We make use of the package ‘lubridate’ to convert the column’s data type to a more appropriate type.

year_data <- year(as.Date(as.character(population_data$Year), 
                     format = "%Y"))

Note: We store the converted data into a new variable so we can reuse it

Store ‘Population’ to a new Variable

Using ‘as.numeric’ to make sure the data is in a correct data type

population <- as.numeric(population_data$Population)

Investigate More

After more investigation, we also notice that the ‘Growth Rate’ column is in a string data type. We can’t directly convert it to double since it includes ‘%’ sign in it. So we need to remove it first. We use the following code:

growth_rate <- gsub('%', '', population_data$`Growth Rate`)

After removing an outlier, we then proceed converting it to ‘double’ data type.

growth_rate <- as.numeric(growth_rate)

Compile the Cleaned Data

Plotting the data

Now that the data is cleaned and ready, we have to find out what the data is telling. The following code is used:

ggplot(data = population_df, aes(x = population, y = year_data)) + 
  geom_point() + 
  geom_smooth(aes(),method = "lm") + 
  labs(x = "Population", y = "Year",title = "Population Growth of the Philippines", 
       subtitle = "1950 - 2023")
## `geom_smooth()` using formula = 'y ~ x'

After visualizing the data, we noticed that the ‘Population’ in the x-axis is hard to read since it is in scientific notation format. So we need to turn it off, and only display the approximate value. The following code is used:

Conclusion and Recommendation

Based on the analysis, it is evident that the population has been experiencing a slow increase over the observed period. Moreover, the population growth rate has been declining steadily, indicating a shift towards a more stable and controlled population growth pattern. This demographic trend has important implications for various aspects of society, including economic development, resource management, and social welfare.

The slow but consistent increase in population indicates the need for long-term planning to accommodate the changing demographic landscape. While the declining growth rate may alleviate some of the immediate pressures associated with rapid population growth, it also raises concerns about potential challenges related to an aging population and workforce.

Addressing the complex dynamics of a slowly increasing population and declining growth rate requires a multi-faceted approach. Policymakers should focus on promoting sustainable development, improving healthcare and education, and creating opportunities for economic growth to ensure a balanced and prosperous society.

Strategies aimed at encouraging family planning, supporting working-age populations, and promoting innovation and productivity can help maintain a healthy and resilient population. By understanding the nuances of this demographic trend and implementing proactive measures, we can ensure a better future for our communities and foster an environment of well-being and progress.
